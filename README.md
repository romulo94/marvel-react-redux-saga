# Test:

Usando a API gratuita da Marvel
https://developer.marvel.com

Iremos listar e mostrar os super-heróis como forma de análise básica do código.

Requisitos WEB

- React WEB
- Usar Reactstrap, Material UI ou outra lib UI de preferência
- Redux
- Redux-saga
- Axios

## Tarefas

- 1 - Em uma onepage listar em cards os heróis da API
- 2 - O card do herói deverá conter imagem e nome
- 3 - Fazer paginação dinâmica, para quando chegar ao fim da tela mostrar mais itens
- 4 - Ao clicar no herói, abrir modal mostrando demais informações vindo da API

## Available Scripts

In the project directory, you can run:

```
  yarn
  yarn start
```
